# Portfolio Website

A modern, responsive portfolio website built with React.

## Features

- Smooth scrolling navigation
- Responsive design for all devices
- Animated sections and transitions
- Contact form
- Project showcase
- Skills visualization
- Social media integration

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

### Build for Production

```bash
npm run build
```

This creates an optimized production build in the `build` folder.

## Customization

1. Update personal information in the components:
   - `Hero.js` - Name, title, and introduction
   - `About.js` - About section and technologies
   - `Projects.js` - Project details
   - `Contact.js` - Contact information
   - `Footer.js` - Footer details

2. Customize colors and styles in the CSS files

3. Replace social media links with your own

## Deployment

You can deploy this portfolio to:
- Vercel
- Netlify
- GitHub Pages
- Any static hosting service
